# experiences/mobile — Expo / React Native experience

The **mobile** channel consuming the one behaviour API through a typed client generated from
the committed OpenAPI contract — the CORS-enabled, cross-origin channel (it talks to the domain
API directly rather than proxying same-origin like web).

Canonical documentation — the typed client, running locally with Expo, and pointing at the
deployed API — lives in
[`docs/experiences/index.md`](../../docs/experiences/index.md#mobile-experiencesmobile).
